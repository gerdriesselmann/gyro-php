= GSitemap Module =

Author: Gerd Riesselmann
Depends: None
Requires: None

== Purpose ==

Build a searchengine XML sitemap 

== Usage ==

When enabled sitemaps are available at /sitemapindex.xml and /sitemap.xml.

If constant APP_GSITEMAP_HTML_SITEMAP is defined as "True", HTML versions 
of the sitemap are avaliable at /sitemapindex.html and /sitemap.html.
