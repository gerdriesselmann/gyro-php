= JQuery Module =

Author: Gerd Riesselmann
Depends: jcssmanager
Requires: Write access to web root

== Purpose ==

Include JQuery (http://jquery.com/) on all pages 

== Usage ==

On install, the module copies jquery.js to the a folder named "js" below web root. This file will
get included on all pages, if JCSSManager Widget is used. It also will be compressed by JCSSManager, 
if compression is enabled.
